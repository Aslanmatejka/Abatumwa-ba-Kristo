## ğŸ”§ Customization Guide

### Update WhatsApp Number

Replace `1234567890` with your actual WhatsApp number in all HTML files:

```html
<a href="https://wa.me/YOUR_NUMBER" class="whatsapp-float"></a>
```

### Change Colors

Edit the CSS variables in `css/style.css`:

```css
:root {
  --primary-color: #e74c3c;
  --secondary-color: #3498db;
  --accent-color: #f39c12;
  /* ... */
}
```

### Add Gallery Images

Replace placeholder images in `projects.html`:

```html
<img src="images/your-image.jpg" alt="Description" />
```

### Update Contact Information

Search for contact details in all files and replace with your actual information.
